# Catch Zone - 게임 기획서

## 1. 개요 (Overview)
**Catch Zone**은 플레이어가 자신의 포즈를 이용해 바구니를 좌우로 움직이며 하늘에서 떨어지는 과일을 받아내는 게임입니다. 폭탄을 피하고, 점수가 높은 과일을 많이 획득하여 최고 점수에 도전하세요.

## 2. 구역 정의 (Zone Definition)
화면은 가로로 3개의 구역으로 나뉩니다.
*   **LEFT (좌측)**: 화면의 왼쪽 1/3 영역
*   **CENTER (중앙)**: 화면의 가운데 1/3 영역
*   **RIGHT (우측)**: 화면의 오른쪽 1/3 영역

## 3. 조작 방식 (Controls)
**Teachable Machine**으로 학습된 포즈 인식 모델을 사용합니다.
*   **왼쪽 포즈 (Left Pose)** → 바구니를 **LEFT** 구역으로 이동
*   **중립 포즈 (Neutral Pose)** → 바구니를 **CENTER** 구역으로 이동
*   **오른쪽 포즈 (Right Pose)** → 바구니를 **RIGHT** 구역으로 이동

## 4. 아이템 종류 (Items)
| 아이콘 | 이름 | 효과 | 비고 |
|:---:|:---:|:---:|:---|
| 💣 | **폭탄 (Bomb)** | **즉시 게임 오버** | 절대 받으면 안 됨 |
| 🍎 | **사과 (Apple)** | **+100점** | |
| 🍐 | **배 (Pear)** | **+150점** | |

## 5. 게임 진행 규칙 (Game Rules)
*   아이템은 화면 상단 랜덤한 구역에서 생성되어 아래로 떨어집니다.
*   플레이어의 바구니가 아이템이 떨어지는 구역과 일치하면 아이템을 **획득**합니다.
*   **획득 판정**:
  - 과일: 점수 획득
  - 폭탄: 게임 오버
*   **놓침(Miss) 판정**: 과일이 바닥에 닿을 때까지 받지 못하면 '놓침'으로 간주합니다. (폭탄은 놓쳐도 무방)

## 6. 게임 오버 조건 (Game Over)
게임은 다음 두 가지 경우에 종료됩니다.
1.  **폭탄을 획득했을 때** (즉시 종료)
2.  **과일을 2번 놓쳤을 때**
    *   1번 놓침: **경고 (Warning)**
    *   2번 놓침: **게임 오버 (Game Over)**

## 7. 레벨 시스템 (Level System)
게임은 20초 단위의 단계(Level)로 진행됩니다.
*   **레벨 지속 시간**: 20초 (게임 오버 없이 버티면 다음 레벨 진입)
*   **낙하 속도 (Drop Time)**:
    *   **1단계**: 2.0초 (아이템이 화면 위에서 아래까지 내려오는 시간)
    *   **레벨 업 효과**: 단계가 오를 때마다 Drop Time이 **0.2초씩 감소** (더 빨라짐)
*   **생성 간격 (Spawn Rate)**:
    *   현재 Drop Time의 **60% ~ 80%** 사이의 무작위 간격으로 아이템 생성

## 8. UI 요소 (UI Elements)
화면에 표시해야 할 필수 정보입니다.
*   **Score**: 현재 획득한 총 점수
*   **Level**: 현재 진행 중인 단계
*   **Time**: 다음 레벨까지 남은 시간 (20초 카운트다운)
*   **Miss/Life**: 남은 기회 표시 (예: ❤️❤️ / 놓침 횟수 0/2)
*   **Game Over Popup**:
    *   "Game Over" 타이틀
    *   최종 점수
    *   "Restart" 버튼
